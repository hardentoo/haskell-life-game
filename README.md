Life Game
==========

Life game implemented by Haskell with [gloss](http://hackage.haskell.org/package/gloss).

```bash
$ git clone git@github.com:lotz84/haskell-life-game.git && cd haskell-life-game
$ stack build && stack exec life-game
```
